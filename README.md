# xml2json

Utility to convert xml to json.[WIP]

## Build

To build run `make` in the current directory.

### Dependencies

Requires `libxml2`.

### Example

./xml2json cust.xml -x"cust.xsd" - convert using XSD schema

./xml2json cust.xml
